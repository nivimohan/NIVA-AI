# NIVA AI – Smart Healthcare Assistant for Senior Citizens

NIVA AI is an intelligent healthcare support platform designed to assist senior citizens with:

- AI-powered health guidance
- Appointment booking
- Medical history tracking
- Emergency contact support
- Simple and accessible UI
